# react hook이란?

> 함수형 컴포넌트에서 react state와 생명주기 기능(lifecycle features)을 연동(hook into)할 수 있게 해주는 함수

- 내장 훅과 커스텀 훅이 있음
- class 안에서는 동작 x

### 규칙

- 반복문, 조건문, 중첩된 함수 내에서 호출X
- 최상위 레벨에서 호출해야함
- react 함수(함수형 컴포넌트) 내에서만 호출

### 대표적인 내장 훅

**1. `useState`**

- 상태(state)값을 추가할 때 사용됨
- 상태가 바뀌면 리렌더링 됨

**2. `useEffect`**

- 화면이 렌더링될 때마다 특정 작업 수행
- sideEffect 처리를 위해 사용됨
