## 1. enumerate

> '열거하다'라는 의미로 iterable한 객체랑 사용할 때 매우 유용한 내장함수다.

```python
arr = ['a', 'b', 'c']
for idx, val in enumerate(arr):
    print(idx, val)
'''
0 a
1 b
2 c
'''
```

enumerate를 사용하면 인덱스에 해당하는 값을 함께 반환해준다.

<br/>

enumerate를 사용하면 간단하게 list를 dictionary 자료형으로 만들 수 있다.

```python
arr = ['a', 'b', 'c']
dic = {val:idx for idx, val in enumerate(arr)}
print(dic)
'''
{'a': 0, 'b': 1, 'c': 2}
'''
```

## 2. zip

> zip은 두개의 리스트를 서로 묶어줄 때 사용한다.

```python

name =  ['a','b', 'c']
score = [10, 20, 30]
for student in zip(name, score):
    print(student)

'''
('a', 10) # 타입은 tuple
('b', 20)
('c', 30)
'''
```

<br/>

zip 내장함수를 활용해서 dictionary를 만들어 보자!

```python
dic = dict(zip(name, score))
'''
{'a': 10, 'b': 20, 'c': 30}
'''

```

```python
name =  ['a','b', 'c']
score = [10, 20, 30]

dic = {name, score for name, score in zip(name, score)}
print(dic)
```

## 3. Counter 객체

`collections` 모듈에서는 `Counter` 객체를 제공한다.<br/>
딕셔너리 형태이고 hashable한 객체를 카운팅하는 데 사용됨<br/>
객체를 key로 하고 개수를 value로 저장 <br/>
데이터 개수를 셀 때 유용<br/>

```python
from collections import Counter
string = 'hello'
print(Counter(string)) # Counter({'l': 2, 'h': 1, 'e': 1, 'o': 1})
```

<br/>

Counter 객체끼리는 뺄셈도 가능함!<br/>
[\[프로그래머스 / Level 1\] 완주하지 못한 선수](https://school.programmers.co.kr/learn/courses/30/lessons/42576?language=python3) Counter로 풀어보기!

```python
from collections import Counter

def solution(participant, completion):
    answer = ''
    dic = Counter(participant) - Counter(completion) # Counter({'kim': 1})
    answer = list(dic.keys())[0]
    return answer
```

해시값이란?<br/>
-> 데이터를 숫자로 변환한 것<br/>
-> 해시 함수에 객체를 넣었을 때 나오는 고유한 정수 값

python에서 hashable 조건 : 아래 두 매직 메서드가 구현되어 있어야 함

- `__hash__()`: 객체를 나타내는 해시값 반환해줌
- `__eq__()`: 다른 객체와 같은지 비교

---

python에서 데이터를 빠르게 찾기 위해서 내부적으로 hash table 사용

왜 어떻게 빠른거지??<br/>
100개 중에서 '사과'를 찾아야 하는 경우<br/>
i. list<br/>
1번 상자부터 100번 상자까지 모두 확인해야함.<br/>
나쁠 경우 100번쨰에서 찾음<br/>
ii. dictionary<br/>
'사과'를 해시 함수에 넣으면 '42번!'이라는 숫자가 나옴<br/>
42번 상자를 곧바로 확인하면 됨<br/>

hash table은 1~100번까지의 사물함이라고 생각<br/>
=> 해시 원리를 실제로 구현하여 데이터를 저장하는 '공간(자료구조)' 그 자체를 말함

---

파이썬 hashable 객체 -> 주로 값이 변하지 않는 객체(immutable)들

- 숫자형: `int`, `float`, `complex`
- 불리언: `bool`
- 문자열: `str`
- 튜플: `tuple` -> 튜플 안에 unhashable한 객체가 하나라도 들어있으면 그 튜플 전체 unhashable
- 불변 셋: `fronzenset`
- 바이너리: `bytes`
- 기타: `None`
