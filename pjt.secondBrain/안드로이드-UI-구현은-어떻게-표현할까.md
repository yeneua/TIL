# 퍼블리싱?-안드로이드-UI-구현은-어떻게-표현할까

다른 팀원이 api 연결 로직을 다 작성해주고 내가 ui 디자인 관련해서 구현하는 태스크를 맡았다.<br/>
지라에 태스크를 작성하려니 '퍼블리싱'이라는 단어가 맞나? 해서 찾아보았다.

## 코틀린에서는...

코틀린에서는 '퍼블리싱'이라는 단어를 잘 쓰지 않음

퍼블리싱은 웹 개발에서 쓰는 용어

퍼블리싱: 디자인 -> HTML/CSS 구현

코틀린은 XML + Compose + 로직이 결합된 환경으로 화면(UI)을 만드는 방식이 웹이랑 다르다

- 웹: HTML(구조) + CSS(디자인)
- 안드로이드
  - XML: 화면 틀(레이아웃)을 정의하는 파일
  - Compose: XML 대신 코틀린 코드로 UI를 만드는 최신 방식
  - 로직(kotlin 코드): 버튼 눌렀을 때 동작, 데이터 처리 등 앱 기능 구현
