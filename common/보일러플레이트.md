# 보일러플레이트(boilerplate)란

> 별도의 수정 없이 반복적으로 사용되는 코드나 문구

매번 프로젝트를 시작하거나 새로운 파일을 만들 때마다 똑같이 작성해야 하는 기본 코드들을 보일러플레이트 코드라고 부름

### 예시

**1. HTML**

HTML 파일을 만들 때마다 항상 쓰는 이 기본 구조가 대표적인 보일러플레이트임

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Document</title>
  </head>
  <body></body>
</html>
```

**2.React**

```jsx
import React from "react";

const ComponentName = () => {
  return <div>{/* 내용 */}</div>;
};

export default ComponentName;
```

**3. Starter Kit**

단순한 코드 조각뿐만 아니라 프로젝트 초기 설정을 미리 다 해놓은 '기초 프로젝트'를 보일러플레이트라고 부르기도 함

ex. 리액트 보일러플레이트 -> react+webpack+ESLint 등 이미 다 세팅된 프로젝트
